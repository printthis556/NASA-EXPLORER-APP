# Contributing to NASA Space Explorer

Thanks for helping improve NASA Space Explorer. This project is kept simple on purpose, so contributions should stay readable, organized, and beginner-friendly.

## How to Contribute

1. Fork the repository on GitHub.
2. Create a branch for your work.
3. Make your changes in a focused way.
4. Test the project locally before opening a pull request.
5. Submit a pull request with a clear description of what changed.

## Contribution Requirements

- Keep the code readable and organized.
- Follow the existing HTML, CSS, and JavaScript structure.
- Test your changes locally before submitting them.
- Document any major behavior changes in the README.
- Use GitHub Issues for bug reports, feature requests, or discussion before making major changes.

## Suggested Local Testing

- Open the project with a local web server.
- Check the main page, gallery, and modal behavior.
- Confirm the site still works on desktop and mobile screen sizes.

## Pull Request Tips

- Keep pull requests small when possible.
- Explain the purpose of the change.
- Include screenshots if the update changes the visual design.
