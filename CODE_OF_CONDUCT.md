# Code of Conduct

We want this project to be welcoming, respectful, and constructive.

Please:

- Be respectful in issues, pull requests, and comments.
- Focus on the project, not the person.
- Give helpful feedback and keep discussions civil.
- Be open to learning and improvement.

Unacceptable behavior includes harassment, personal attacks, discrimination, and disruptive conduct.

If a problem happens, please raise it privately with the repository owner.
